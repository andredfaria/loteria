Qual critério de cor faz mais sentido para as dezenas?

Cada opção usa um critério diferente para colorir os números 1–25. Clique na que preferir.

A — Frequência histórica (quente / frio)

Cor baseada em quantas vezes cada número foi sorteado nos últimos 500 concursos. Verde = mais sorteado, vermelho = menos sorteado, azul = médio.

As cores mudam a cada carregamento conforme os dados reais do banco.

B — Grupos de 5 dezenas

Divide o intervalo 1–25 em 5 faixas de igual tamanho, cada uma com uma cor distinta. Fácil de ler, estável, sem dependência de dados.

Cada número recebe cor conforme sua natureza matemática. Útil porque os filtros estatísticos da lotofácil (primos, fibonacci, par/ímpar) já usam esses conceitos.
